# FinApp

Aplicativo mobile de controle financeiro desenvolvido com **React Native**, **Expo** e **Supabase**.

## Funcionalidades

- Cadastro de receitas e despesas
- Controle do saldo mensal
- Metas financeiras
- Histórico por mês
- Organização por categorias

## Tecnologias

- React Native
- Expo
- TypeScript
- Supabase (Auth + Banco de dados)
- PostgreSQL

## Como executar

```bash
npm install
npx expo start
